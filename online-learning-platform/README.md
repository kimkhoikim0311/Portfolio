# 🎓 Online Learning Platform (Edumine)

---

## Project Overview

### Background

Edumine is an online learning platform designed to support effective teaching and learning in a digital environment. The platform bridges the gap between teachers and learners by providing tools that make communication, progress tracking, and course management simple and intuitive.

### Project Goal

To create an online learning environment where teaching and learning can take place naturally and effectively without the need for a traditional classroom.

---

## My Role

UI/UX Designer

---

## Design Process

### 01 — Discover
Focused on understanding user needs and pain points in online learning environments. Research helped identify common challenges faced by both teachers and students.

**Deliverables:**
- User research
- Competitor audit

### 02 — Define
Based on research insights, defined core features and user flows to address real user problems. This phase helped clarify product scope and structure.

**Deliverables:**
- Function & feature list
- Site map
- User flow
- Demo concept

### 03 — Design
Focused on creating a clean, modern interface with a strong emphasis on usability and consistency across all user flows.

**Deliverables:**
- Style guideline
- Component library
- UI design
- Responsive design
- Dark mode

### 04 — Testing
Testing was conducted to evaluate usability and identify areas for improvement. Feedback was used to refine interactions and improve user experience.

**Deliverables:**
- Heuristic evaluation
- Revision
